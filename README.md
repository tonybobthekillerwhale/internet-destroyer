# Internet Destroyer Chrome Extension

Blow up the internet. 💥

## Install

1. `git clone https://github.com/tonybobthekillerwhale/internet-destroyer.git`
2. Go to: `chrome://extensions/` and activate Developer mode.
3. Click `Load unpacked` and open the `internet-destroyer` folder.

## Usage

Click the toolbar icon for the popup menu.

`Ctrl+Delete` toggles target mode.

If the shortcut does nothing, check `chrome://extensions/shortcuts`.

### Settings (Popup Menu)

**Highlight border/glow** - Toggles border and glow on targeted elements

**Highlight background** - Toggles background highlighting on targeted elements

**Undo Last Destruction** - Restores the last destroyed element

**Clear Destructions (this page)** - Clears destroyed elements on the current page

**Clear ALL Destructions (everywhere)** - Clears destroyed elements across all pages visited

## Known Issues

* Some elements won't remain destroyed beyond refresh due to dynamic element naming/generation.
